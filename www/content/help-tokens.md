## Tokens

Placeholder. Each token is a single vowel measurement from a single word. Showing tokens lets you see the raw spread of the data before any summarization.

### Points vs. words

- **Points** — a small dot at the F1/F2 midpoint of each token. Use when you have many tokens and want to see density.
- **Words** — the orthographic word label plotted at each token's midpoint. Useful for spotting outliers or checking which words are driving the distribution.

### Opacity and size

Reduce opacity when tokens overlap heavily. Reducing size helps with dense plots.
