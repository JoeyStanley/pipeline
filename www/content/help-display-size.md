## Display size

Controls the dimensions and resolution used when rendering the plot in the browser. These are independent of the **Download** tab settings, which control the exported file.

- **Width / Height** — rendered size in inches. Larger values give more room for labels and reduce overplotting, but may extend below the visible area of the sidebar.
- **DPI** — dots per inch. Higher DPI produces a crisper image at the cost of a larger file in memory. 150 DPI is a reasonable default for screen display; 300 DPI is standard for print-quality exports (set that in the Download tab instead).
