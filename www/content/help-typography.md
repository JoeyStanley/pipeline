## Typography

Control the base font size and font family used throughout the plot. These map to ggplot2's `base_size` and `base_family` theme arguments.

### Base font size

All text in the plot (axis labels, tick marks, legend, title) scales relative to this value. Larger values are better for presentations; smaller values work well for journal figures with many panels.

### Font family

The font must be installed on your system for it to render correctly. If a chosen font is unavailable, ggplot2 will fall back to its default. Common safe choices are **Helvetica** and **Times** (available on most systems). **Avenir** and **Palatino** are macOS defaults and may not render on Windows or Linux.
